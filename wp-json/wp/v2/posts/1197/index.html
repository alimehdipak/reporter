A clever iPhone hack lets you type some texts much faster.<\/p>\n

Better yet, it works on iMessage and\u00a0WhatsApp\u00a0too \u2013 so you can seriously upgrade your texting game.<\/p>\n

Texting can take up a lot of time, especially if you\u2019re sending a very long message on\u00a0iPhone.<\/p>\n

But if you regularly send similar texts, there\u2019s an easy solution.<\/p>\n

It\u2019s possible to use Apple\u2019s text-replacement feature to quickly type out long missives.<\/p>\n

So if you often share an address, a phone number, a common phrase or anything else regularly, it\u2019s worth setting up a text-replacement for it.<\/p>\n

The hack was recently shared by\u00a0TikTok\u00a0creator\u00a0@ericnowoslawski, earning hundreds of likes.<\/p>\n

In the video, Eric uses the feature to quickly share personal info, like a personal email address.<\/p>\n

And he puts the letter \u201cx\u201d before a phrase \u2013 like \u201cxemail\u201d \u2013 to trigger the text replacement.<\/p>\n

This is something you wouldn\u2019t ordinarily type, so it\u2019s safe to use it as a text replacement.<\/p>\n

And it can save you loads of time.<\/p>\n

The hack was recently shared by TikTok creator @ericnowoslawski, earning hundreds of likes.TikTok\/@ericnowoslawski <\/p>\n

How to set up iPhone text replacement<\/h2>\n

First, open Settings and then go into General.<\/p>\n

Scroll down to Keyboards and then tap into Text Replacement.<\/p>\n

You can then create replacements, typing in a phrase and a shortcut.<\/p>\n

In the video, Eric uses the feature to quickly share personal info, like a personal email address.TikTok\/@ericnowoslawski <\/p>\n

The shortcut is what you personally type out, and the phrase is what it transforms into.<\/p>\n

So keep the shortcut short, and have the phrase as long as you\u2019d like.<\/p>\n

Now when you type that shortcut using the built-in\u00a0Apple\u00a0keyboard, it\u2019ll automatically transform into the phrase.<\/p>\n

It\u2019s possible to use Apple\u2019s text-replacement feature to quickly type out long missives.TikTok\/@ericnowoslawski <\/p>\n

Make sure to pick shortcuts that are slightly obscure that you wouldn\u2019t accidentally type in normal conversation.<\/p>\n

If one becomes annoying or isn\u2019t very useful, you can edit or delete it at any time.<\/p>\n
