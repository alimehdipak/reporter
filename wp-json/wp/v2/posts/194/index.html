    \n
  • QuickBooks lets you easily import suppliers and vendors from other sources in bulk.<\/li>\n
  • You can transfer vendor data from bookkeeping software, Excel files and elsewhere.<\/li>\n
  • Ensure you’ve collected all pertinent vendor details, such as their name, employer identification number and rates.<\/li>\n
  • This article is for small business owners and bookkeepers getting started with QuickBooks to manage finances.\u00a0<\/li>\n<\/ul>\n

    QuickBooks has powerful features that make it simple for business owners, or their accountants or bookkeepers, to handle all aspects of business accounting. With QuickBooks’ integrated vendor management function, you can easily track vendors, payables and other relevant details.<\/p>\n

    While you can manually set up vendors in QuickBooks, it’s much easier to import existing vendor data in bulk. This guide will show you how to import vendor information from Excel and CSV files into whichever version of QuickBooks you’re using, including QuickBooks Online and QuickBooks Pro.<\/p>\n

    Tip: If you have more complex finance and accounting needs, consider the features of QuickBooks Online Advanced, which includes smart reporting and the ability to enter invoice transactions from a CSV file.<\/p>\n

    Adding vendors to QuickBooks Online<\/h2>\n

    If you’re using the cloud-based QuickBooks Online application, accessible from a web browser, it’s easy to add and import vendors.\u00a0<\/p>\n

    With all your vendor information in QuickBooks Online, you’ll save time when entering bills and transactions. Instead of entering details manually, you’ll select a vendor from a drop-down list with many fields already filled. It’s also easy to track expenses through accounting reports, providing key insights to managers and business owners.<\/p>\n

    QuickBooks Online vendor setup checklist<\/h3>\n

    Whether you’re entering vendors manually or importing them, it’s essential to collect your vendors’ important information.\u00a0<\/p>\n

    It’s easy to gather information in a CSV file or Excel spreadsheet. If you’re using another accounting software program to manage your suppliers, you can transfer details to QuickBooks easily.<\/p>\n

    You should have the following information for all your vendors:<\/p>\n

      \n
    • Company name<\/li>\n
    • Email address<\/li>\n
    • Phone number<\/li>\n
    • Contact name<\/li>\n
    • Employer identification number<\/li>\n
    • Outstanding balance before importing into QuickBooks<\/li>\n
    • Rate paid to the vendor<\/li>\n
    • Rate charged from clients for services<\/li>\n<\/ul>\n

      Manual vendor setup vs. bulk imports in QuickBooks Online<\/h3>\n

      If you have only a few vendors to add to QuickBooks Online, it makes sense to enter them manually. Here’s how it works:\u00a0<\/p>\n

        \n
      1. Select Expenses > New Vendor.\u00a0<\/li>\n
      2. In the Vendor Center, select New Vendor.\u00a0<\/li>\n
      3. Fill out your vendor information, and select Save when you’re done.\u00a0<\/li>\n
      4. Edit the details anytime by selecting the vendor and then selecting Edit. \u00a0<\/li>\n<\/ol>\n

        If you have more than a few vendors, it’s much easier to import them into QuickBooks Online using an existing vendor list in Excel or other bookkeeping software.\u00a0<\/p>\n

        Did you know? QuickBooks also has powerful payroll and time-and-attendance tools for small businesses. Read our QuickBooks Payroll review and QuickBooks Time review for more information.<\/p>\n

        Import vendors to QuickBooks Online using a CSV or Excel file<\/h2>\n

        There are a few steps to importing your vendor data into QuickBooks online via CSV or Excel file. You’ll need to prepare your spreadsheet and then initiate and complete the import in QuickBooks Online.\u00a0<\/p>\n

        Step 1: Prepare the spreadsheet.<\/h3>\n

        Every spreadsheet column equates to a field you want to import into QuickBooks Online, and each row contains details about an individual vendor.\u00a0<\/p>\n

        QuickBooks allows you to import the following fields, but you don’t have to have data for every field:\u00a0<\/p>\n

          \n
        • Company name<\/li>\n
        • Contact name<\/li>\n
        • Phone number<\/li>\n
        • Email address<\/li>\n
        • Mobile number<\/li>\n
        • Website<\/li>\n
        • Fax number<\/li>\n
        • Street address<\/li>\n
        • City<\/li>\n
        • ZIP code<\/li>\n
        • State<\/li>\n
        • Country<\/li>\n
        • Tax identification number<\/li>\n
        • Opening balance<\/li>\n
        • Date of opening balance<\/li>\n<\/ul>\n

          Your spreadsheet should contain the labels for each of the columns in the first row. However, these label names don’t have to be exactly the same as the field names in QuickBooks Online. The import process gives you a chance to map each column to a different field on the vendor list. Avoid the columns for fields you don’t wish to populate.<\/p>\n

          The vendor information spreadsheet must be on the first workbook tab in Excel. QuickBooks Online will consider only the first tab.\u00a0<\/p>\n

          <\/p>\n

          Tip: If QuickBooks isn’t right for your business, consider some feature-rich and affordable QuickBooks alternatives. Read our review of Xero, our Sage review and our Zoho Books review to learn more.<\/p>\n

          Step 2: Import the file into QuickBooks Online.<\/h3>\n
            \n
          1. Launch QuickBooks Online and select Expenses > Vendors.\u00a0<\/li>\n
          2. In the Vendor Center, go to the New Vendors drop-down arrow and select Import Vendors.\u00a0<\/li>\n
          3. In the box labeled Select a CSV or Excel file to upload, select Browse.<\/li>\n
          4. Navigate to your CSV or Excel file and click Open.<\/li>\n<\/ol>\n

            <\/p>\n

              \n
            1. Select Next to continue.\u00a0<\/li>\n
            2. On the Map Data page, map your fields from the imported file to QuickBooks Online fields. If your column headers match QuickBooks Online’s fields, the software makes a selection automatically. These are just suggestions; you can always make changes.\u00a0<\/li>\n
            3. It’s OK if your imported sheet has extra headers, and you don’t need to have matches for all the software’s vendor columns. Select No Match if you don’t have a match. Select Next when you’re ready to continue.<\/li>\n<\/ol>\n

              <\/p>\n

                \n
              1. Review your data. Confirm that the number of imported records equals the number of rows in your spreadsheet, excluding the header row. If you see any issues, correct them here or select Back to make any necessary mapping changes. Deselect any records you don’t want to import.<\/li>\n
              2. When you’re satisfied, select Import to proceed.\u00a0<\/li>\n<\/ol>\n

                <\/p>\n

                  \n
                1. When the import completes, you’ll see a message about the vendors you successfully imported. Your vendors are now available in QuickBooks Online’s Vendor Center.\u00a0\u00a0<\/li>\n<\/ol>\n

                  Tip: Read our complete review of QuickBooks Online accounting software to learn more about the powerful features of this cloud-based tool.<\/p>\n

                  How to import vendors into QuickBooks Pro<\/h2>\n

                  It’s also fast and easy to import your vendors into QuickBooks Pro, the QuickBooks accounting desktop application.\u00a0\u00a0<\/p>\n

                  Like QuickBooks Online, QuickBooks Pro lets you add supplier details manually or import a spreadsheet CSV to add vendors in bulk.\u00a0<\/p>\n

                  Here’s how to import vendors from an Excel spreadsheet into QuickBooks Pro.<\/p>\n

                  Step 1: Select your vendors.\u00a0<\/h3>\n
                    \n
                  1. Launch QuickBooks Pro and select Vendors to go to the Vendor Center.\u00a0<\/li>\n
                  2. Select Excel > Import From Excel.<\/li>\n
                  3. Under Select the type of data you want to add to QuickBooks, select Vendors.\u00a0<\/li>\n
                  4. You’ll be taken to a screen with a formatted Excel sheet. Skip this step if you’re using an Excel file in the same format as the QuickBooks Pro Vendor Import Template.\u00a0<\/li>\n
                  5. Select Advanced Import to complete your import.\u00a0<\/li>\n
                  6. You’ll see a pop-up warning that you can’t undo an import. Click Yes to continue.<\/li>\n<\/ol>\n

                    Step 2: Get your vendor data.<\/h3>\n

                    Copy the data from your Excel vendor file to the QuickBooks template. Your Excel data is on the right, and the QuickBooks template is on the left. Hovering over a cell displays instructions on which type of data you should add. Above the column headers, you’ll see cell-formatting examples.\u00a0<\/p>\n

                    QuickBooks lets you add the following vendor information:\u00a0<\/p>\n

                      \n
                    • Company name<\/li>\n
                    • First name<\/li>\n
                    • Last name<\/li>\n
                    • Street address<\/li>\n
                    • City<\/li>\n
                    • State<\/li>\n
                    • ZIP code<\/li>\n
                    • Country\u00a0<\/li>\n
                    • Phone number<\/li>\n
                    • Alternate number<\/li>\n
                    • Fax number<\/li>\n
                    • Account number<\/li>\n
                    • Email address<\/li>\n<\/ul>\n

                      Step 3: Resolve any errors.<\/h3>\n

                      When your file is ready to be imported, you’ll have a chance to check for and correct errors. If the program finds any issues, you’ll see a message in the error box on the top left and a red warning triangle in the corner of the affected cell. If there’s an error, QuickBooks will explain it and tell you how to fix it.\u00a0<\/p>\n

                      Step 4: Save your changes.<\/h3>\n
                        \n
                      1. After verifying that everything is correct, select Add My Data Now to populate the vendor information into QuickBooks Pro.\u00a0<\/li>\n
                      2. You’ll see an error message warning that you haven’t yet saved your changes. Select Save & Add Data Now.<\/li>\n
                      3. Save the file to your computer. You should see the filename automatically in the box. If not, select Browse and select the file.<\/li>\n<\/ol>\n

                        Step 5: Import the file.<\/h3>\n
                          \n
                        1. Select Add My Data Now. You’ll see a screen with all your successfully completed steps along with the number of records added to your vendor list.\u00a0<\/li>\n
                        2. Your vendor import is complete. Go to the Vendor Center to view your vendor list.\u00a0\u00a0<\/li>\n<\/ol>\n

                          How to delete a vendor in QuickBooks<\/h2>\n

                          You may want to delete a vendor at some point. However, if you’ve already used a vendor’s details in a transaction, you can’t delete the vendor from QuickBooks. You can, however, make the supplier inactive so you won’t see it in the Vendor Center.\u00a0\u00a0<\/p>\n

                          To make a vendor inactive:<\/p>\n

                            \n
                          1. Navigate to the Vendor Center.<\/li>\n
                          2. Find the vendor you want to deactivate.<\/li>\n
                          3. Select the drop-down menu and choose Make Inactive.\u00a0<\/li>\n
                          4. Select Yes to confirm.\u00a0<\/li>\n
